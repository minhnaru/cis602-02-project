CDC DATA IN THE UNITED STATES FROM 2013 TO 2015

ABOUT

Natality Information
This is an interactive web application designed to help users visualize spatial and statistical relationships between births, fertility, infant death, median income, and tobacco use.

Manual
1. Use the dropdown menus "Slect a Visualization" to view different variables on the choropleth map and scatterplot.
2. Use the dropdown menus "Select a Data" to show the relationships between two datasets in scatterplot and the changes of a State in the line chart (2013-2015).
3. Hover the map to see more information, and click on a State to see the change in the line chart.
